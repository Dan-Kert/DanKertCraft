## ✨ Key Features
- ✅ **Cross-platform**: Seamless support for both Windows and Linux.
- ✅ **Java Downloader**: Automatically detects and downloads the required Java environment version.
- ✅ **Mod Support**: Fully integrated with Modrinth for easy modpack installations.
- ✅ **Version Management**: Support for installing and running various versions of Minecraft.
- ✅ **Localization**: Out-of-the-box availability in both Russian and English.

## 📋 Requirements
- Java 17+
- JavaFX 17+

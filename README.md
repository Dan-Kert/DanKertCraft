## ✨ Основные возможности
- ✅ **Кроссплатформность**: Windows и  Linux
- ✅ **Скачивание Java**: скачивание нужной версии
- ✅ **Поддержка модификаций**: интеграция с Modrinth
- ✅ **Управление версиями**: поддержка разных версий Minecraft
- ✅ **Локализация**: русский и английский языки

## ✨ Key Features
- ✔ Cross-platform: Windows and Linux support.
- ✔ Java Downloader: Automatically downloads the required Java version.
- ✔ Mod Support: Seamless integration with Modrinth.
- ✔ Version Management: Support for all Minecraft versions.
- ✔ Localization: Available in Russian and English.

## 📋 Требования
- Java 17+
- JavaFX 17+
